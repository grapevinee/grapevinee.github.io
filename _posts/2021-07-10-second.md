# 내 컴퓨터 bit 확인
1. Win + E 눌러 파일 탐색기 열기
2. 내PC 오른쪽 마우스 클릭 - 속성
3. 시스템종류에서 비트 확인

# 내 컴퓨터 IP 확인
1. Win + R 눌러 실행창 열기
2. cmd 입력해 실행하기 **cmd:command(컴퓨터로 시스템이나 장치에 대하여 특정 기능의 실행을 지시하기 위한 정보나 신호)의 약자
3. ipconfig 입력 후 엔터
4. IPv4주소에 나온 숫자가 내 컴퓨터 IP
**192.168로 시작하면 사설 IP (공유기 연결 PC라면 사설IP가 부여된다)
